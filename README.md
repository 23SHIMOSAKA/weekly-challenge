# About
Geminiに週一回、お題を出してもらう  
参加者はお題に沿った写真を投稿  
GASでの運用を想定
# Usage
1. GeminiのAPIキーを発行（無料のやつあり）  
1. GASで.gsファイルにこれをコピペ
2. APIキーや投稿先のwebhook URLを設定（専用のフォーラムやスレッドがおすすめ）
3. 必要に応じてプロンプトを調整、現状うまく動く
# Note
- 週一回ということで、その週にちなんだお題を出させている（ただし微妙なときも多い）
- デバッグ等足りてない、特にエラー処理とか雑なので動かない場合あるかも？
- 時間ベースのトリガー設定しとくといいかも
